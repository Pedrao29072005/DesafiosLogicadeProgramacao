# Logica-de-programacao-praticando-com-desafios
Curso da Alura para reforçar os conhecimentos em Lógica de Programação. Para o reforço das ideias foram pensados 5 desafios, cada um deles com a mesma estrutura básica, mas claro ficando cada vez mais difícil.

# 1° Desafio: Sorteador de Números
O primeiro desafio proposto é algo mais sim´ples, tendo que ser feito um jogo capaz de sortear números em um interivalo especificado pelo úsuario com base na estrutura disponibilizada previamente no arquivo HTML.
# 2° Desafio: Alugames
Nesse desafio será necessário alterar os status de um jogo dentro do sistema de aluguel de jogos, se ele estiver alugado a opção no botão deve ser DEVOLVER, e se ele estiver com a opção de DEVOLVER habilitada a opção deve mudar para ALUGAR ao clicar novamente o botão.
# 3° Desafio: Carrinho de Compras
Selecionar um determinado produto, informar a quantidade, adicionar ao carrinho, realizar a soma de todos os produtos que estão no carrinho, além de também ter a opção de zerar o valor e tirar todos os itens da lista.
# 4° Desafio: Ingresso Online
A proposta desse desafio é simular a funcionalidade de compra de ingressos, a lógica de compra de ingressos é a mesma pra os 3 tipos citados no desafio, só muda o nome da função que manipula cada tipo de ingresso.
# 5° Desafio: Amigo secreto
O desafio consiste em implementar um sorteador de amigos, em uma brincadei9ra de amigo secreto, podendo incluir várias pessoas, realizando vários sorteios diferentes.
